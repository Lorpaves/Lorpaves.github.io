Basic Macos Terminal Commands

Posted Feb 8, 2021

 1$ ls   # list
 2$ pwd  #  present work directory
 3$ clear # clean all lines
 4$ cd  <directory> # change directory
 5$ cd ..  # back to the directory
 6$ cd  # to the root directory
 7$ cd ~  # to the user directory
 8$ open . # open current folder
 9$ mkdir <folder_name>  # make folder
10$ touch <file_name>   # make file
11$ open <file_name>   # open file
12$ open -a "application_name" <file_name>   # open file in application
13$ mv <origin_file_name <new_file_name>  #  rename file
14$ rm <file_name>  # remove file
15$ rm -r <folder_name>  # remove folder
16$ say <something>   # say something
