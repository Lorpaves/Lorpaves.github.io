如何在MacOS上彻底地卸载Python

问题:

如果你现在的Python的版本存在问题（比如Python的3.7版本）然后你想下载另一个版本（例如3.9最新版本）。为了下载一个全新的版本你需要先卸载当前的版本。

解决方法:

假定当前电脑的Python版本为3.7，现在要用已经下载的3.9（新的）版本替换掉3.7（原先的）版本。

按照以下三步来

  1. 移除Python3.7第三方构架(framework of the third-party)

    1$ sudo rm -rf /Library/Frameworks/Python.framework/Versions/3.7

  2. 移除应用里的Python目录

    1$ sudo rm -rf "/Applications/Python 3.7"

  3. 移除指向现在版本的Python的符号链接（symbolic links），在 /usr/local/bin 里

    1$ ls -l /usr/local/bin | grep '../Library/Frameworks/Python.framework/Versions/3.7'

    然后输入以下命令来移除所有链接:

    1$ cd /usr/local/bin/
2$ ls -l /usr/local/bin | grep '../Library/Frameworks/Python.framework/Versions/3.7' | awk '{print $9}' | tr -d @ | xargs rm

下载新的版本:

    1. 在官网页面，选择要下载的版本进行下载安装。
    1. 验证是否安装成功: $ python3 -version
    1. 安装Homebrew
    1. 安装 virtualenv
    1$ pip3 install virtualenv
2$ pip3 install virtualenvwrapper
    1. 创建并激活Python的Virtual Environment
    1$ cd /Users/admin/Downloads/training_model/model 
2$ python3 -m virtualenv /Users/admin/Downloads/training_model/model
3$ source bin/activate
